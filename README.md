CT-BLOG
=======

## 前期准备

安装Nodejs 12

## 启动
```shell script
npm run start
```

## 发布到github page
```shell script
npm run deploy
```

## 博客书写规范
必须添加文件头
```shell script
---
date: "2021-04-21"
title: "网站草图"
category: "tool"
tag: ""
type: "学习整理 | 问题解决 | BUG复盘 | 其他"
author: "github账号名"
---
```
